## ğŸš€ Quick Start

1. Download `excel/Coop_Management.xlsx`.
2. Open in Excel.
3. Start by entering your members in the **Members** sheet.
4. Record weekly/monthly contributions in **Contributions** / **Weekly_Contributions**.
5. Issue loans in the **Loans** sheet.
